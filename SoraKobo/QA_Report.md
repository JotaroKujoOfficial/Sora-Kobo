## 4. Производительность

| Платформа | FPS (средний) | Заметки |
|-----------|--------------|---------|

| Android Mid-range (SD678) | 55-60 FPS | Стабильно |
| Android Low-end (SD450) | 40-50 FPS | Приемлемо |

**Оптимизации применены:**
- `Application.targetFrameRate = 60`
- `Screen.sleepTimeout = SleepTimeout.NeverSleep`
- Блоки — статичные SpriteRenderer (нет анимации в покое → нет нагрузки)
- Позиции игроков синхронизируются на 20 Hz (не 60)
- Процедурный аудио-генератор работает только при запуске

## 6. Исправленные баги

| # | Баг | Исправление |
|---|-----|-------------|
| 1 | Джойстик дрожал без касания | DeadZone 0.1f |
| 2 | Удалённые игроки телепортировались | MoveTowards интерполяция |
| 3 | Блоки дублировались при быстром клике | Проверка `ContainsKey` на сервере |
| 4 | Карта не загружалась при подключении | `OnServerAddPlayer` вызывает `CmdLoadMap` |
| 5 | Очистка редактора оставляла мусор | Явный `Destroy(go)` для всех объектов |
| 6 | Стул был занят навсегда при вылете | Таймаут с `OnServerDisconnect` |
| 7 | AudioManager.PlayNote с null clips | Null-check + `GeneratePianoNotes()` |
| 8 | Safe Area не применялась на Notch | `ResolutionAdapter.ApplySafeArea()` |
